Это демо простого web-приложения с использованием:
  • JavaScript, Ajax
  • Servlet
  • JDBC
  • JasperReports
  • REST
  • SOAP
  • MQ
Включает следующие функции:
  • Аутентификацию
  • Сохранение пользовательской информации в cookies
  • Аудит
  • Просмотр списка клиентов
  • Создание клиента
  • Редактирование клиента
  • Просмотр списка заявок
  • Создание заявки
  • Отмена заявки
  • Формирование отчета по заявкам
  • Работа с заявками через REST
  • Работа с заявками через SOAP
  • Работа с заявками через MQ
  • Ведение интеграционного лога
  • Выполнение автоматического задания по обработке заявок (отмена)
